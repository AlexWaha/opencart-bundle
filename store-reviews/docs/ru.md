## 2. Ключевые возможности

- Отзывы о магазине (отдельно от отзывов о товарах)
- Полное управление в админке (добавление, редактирование, удаление)
- Отдельная страница отзывов с пагинацией
- AJAX-форма отправки отзывов для покупателей
- Звёздный рейтинг (1-5)
- Поля: имя автора и город
- Модерация отзывов (переключатель статуса)
- Виджет-карусель отзывов (для главной, сайдбара и т.д.)
- Карусель на Swiper.js (в комплекте, CDN не нужен)
- Настраиваемое кол-во слайдов с адаптивными точками останова
- SEO URL и мета-теги для каждого языка с валидацией уникальности
- Импорт/Экспорт настроек в JSON
- Интеграция с AW Microdata (Schema.org AggregateRating + Review)
- Интеграция меню в админку через OCMOD

## 3. Установка

### Предварительные требования

- OpenCart 2.3 - 3.x
- Установленная библиотека AwCore
- PHP 7.4+

### Шаги установки

1. Перейдите в **Расширения - Установщик расширений**
2. Загрузите `aw_store_reviews_oc2.3-3.x.ocmod.zip`
3. Перейдите в **Расширения - Модификации** и нажмите **Обновить**
4. Перейдите в **Расширения - Модули**
5. Найдите **alexwaha.com - Store Reviews** и нажмите **Установить** (создаст таблицу `aw_review`)
6. Найдите **alexwaha.com - Store Reviews Carousel** и нажмите **Установить**
7. Настройте оба модуля

## 4. Store Reviews - Настройка

### 4.1 Список отзывов

Главная страница модуля в админке отображает все отзывы в таблице:
- Автор, Город, Рейтинг (звёзды), Статус, Дата, Действия (редактировать/удалить)
- Фильтрация по статусу
- Массовое удаление

### 4.2 Добавление/Редактирование отзыва

- **Автор** - Имя рецензента
- **Город** - Город рецензента (необязательно)
- **Текст** - Текст отзыва
- **Рейтинг** - Звёздный рейтинг 1-5 (интерактивный выбор)
- **Статус** - Включить/выключить (модерация)

### 4.3 Вкладка "Настройки"

- **Статус** - Глобальный статус модуля
- **Отзывов на странице** - Лимит пагинации
- **SEO URL** - Ключевое слово URL для каждого языка (с проверкой уникальности)
- **Мета-заголовок** - Заголовок страницы для SEO (для каждого языка)
- **Мета-описание** - Описание страницы для SEO (для каждого языка)
- **Мета H1** - Пользовательский заголовок H1 (для каждого языка)

### 4.4 Импорт / Экспорт

- **Экспорт** - Скачать настройки в формате JSON
- **Импорт** - Восстановить настройки из JSON

## 5. Store Reviews Carousel - Настройка

Карусель - это отдельный виджет-модуль (как стандартный модуль "Рекомендуемые" в OpenCart). Можно создать несколько экземпляров и назначить их на разные позиции макета.

### Настройки экземпляра

- **Название** - Внутреннее имя экземпляра
- **Заголовок** - Текст заголовка карусели (для каждого языка)
- **Отзывы** - Выбор отзывов через автодополнение, перетаскивание для сортировки
- **Лимит (если не выбраны)** - Кол-во последних отзывов, если не выбраны вручную
- **Отзывов на слайд** - Сколько отзывов показывать одновременно на десктопе (планшет: макс. 2, мобильный: 1)
- **Статус** - Включить/выключить экземпляр

### Возможности карусели

- На базе [Swiper.js](https://swiperjs.com/) (в комплекте, без CDN)
- Стрелки навигации (вперёд/назад) с эффектом при наведении
- Точечная пагинация (кликабельная)
- Режим loop — бесконечная прокрутка
- Автопрокрутка каждые 5 секунд с паузой при наведении
- Поддержка свайпа на мобильных
- Адаптивные точки останова (настраиваются через "Отзывов на слайд")
- Ссылка на полную страницу отзывов

## 6. Страницы на сайте

### Страница отзывов

Доступна по настроенному SEO URL (например, `/otzyvy/`). Отображает:

- Список одобренных отзывов со звёздным рейтингом, автором, городом, датой
- Пагинация
- Форма "Оставить отзыв": имя, город, выбор рейтинга, текст, кнопка отправки
- AJAX-отправка с сообщениями об успехе/ошибке
- Отзывы появляются после модерации администратором

### Виджет-карусель

Показывает выбранные отзывы в слайдере. Появляется в позиции макета, куда назначен экземпляр модуля (главная, сайдбар, подвал и т.д.).

## 7. Решение проблем

| Проблема | Решение |
|----------|---------|
| Страница отзывов выдаёт 404 | Задайте SEO URL в настройках, очистите кеш SEO |
| Карусель не отображается | Проверьте что экземпляр модуля включён и назначен на позицию макета |
| Новые отзывы не видны | Проверьте статус - новые отзывы требуют одобрения администратором |
| Стрелки карусели отсутствуют | Убедитесь что шаблон загружается (проверьте консоль браузера) |
| Меню OCMOD не видно | Обновите кеш модификаций |
| Ошибка SEO URL при сохранении | Для каждого языка URL должен быть уникальным |
