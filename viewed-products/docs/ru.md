# Alexwaha.com - Viewed Products

> **Живое демо:** посмотреть модуль на демо-витрине - [demo.alexwaha.com/module-viewed-products](https://demo.alexwaha.com/module-viewed-products)

**Модуль для OpenCart v2.3 - 3.x**

---

## Описание

Viewed Products показывает каждому посетителю товары, которые он недавно просматривал. Каждый просмотр страницы товара отслеживается автоматически, а история переносится из гостевого cookie в аккаунт покупателя после входа в систему.

Модуль состоит из двух частей на витрине:

1. **Блок «Просмотренные товары»** - виджет с возможностью размещения в макете (можно добавить на страницу товара, главную или в любое место через Дизайн -> Макеты). Выводится в стиле карточек родной темы и всегда загружает товары через **AJAX** - это исключает отравление кеша полных страниц и предотвращает показ чужой истории. Текущий открытый товар исключается автоматически.
2. **Страница аккаунта «Просмотренные товары»** - отдельная страница в аккаунте покупателя (`Viewed Products`), отображающая полную историю с пагинацией. Также загружается через AJAX (single mode - устаревший режим серверного рендеринга удалён для повышения производительности), и исключена из индексирования с помощью тега robots `noindex`.

Авторизованный покупатель может удалить любой товар из истории (кнопка удаления на каждой карточке). История гостя объединяется с аккаунтом покупателя при входе.

## Инструкция по установке

1. **Установка основного модуля**

   Убедитесь в том, что у Вас установлен модуль **[aw_core_oc2.3-3.x.ocmod.zip](https://github.com/AlexWaha/opencart-bundle/blob/main/Core/dist/aw_core_oc2.3_3.x.ocmod.zip)**

   > Это вспомогательный функционал для работы всех моих модулей. **Критически необходимо**, без него работать модуль не будет

2. **Настройки FTP-загрузки (только для OpenCart 2.x)**

   Убедитесь, что загрузка дополнений по FTP разрешена (или настройки FTP корректны) на OpenCart 2.x. **(Для OpenCart 3.x не требуется)**

3. **Установите файлы модуля aw_viewed_oc2.3-3.x.ocmod.zip**
4. **Включение модуля**

   В разделе: **Расширения -> Модули** -> Установите и откройте настройки модуля **alexwaha.com - Viewed Products**

   > При установке модуль создаёт свою таблицу в БД, выдаёт права на просмотр и редактирование (текущей группе пользователей и группе Administrator) и регистрирует события витрины. Обновление кеша модификаций не требуется - модуль работает через систему событий OpenCart.

5. **Размещение виджета (необязательно)**

   Перейдите в **Дизайн -> Макеты**, откройте нужный макет (например, Товар) и добавьте модуль **alexwaha.com - Viewed Products** в нужную позицию (например, Content Bottom).

### Самопроверка после установки

- Откройте **Расширения -> Модули -> alexwaha.com - Viewed Products** - откроется форма настроек с вкладками Widget / Tracking / Account Page
- Просмотрите несколько страниц товаров как гость, затем откройте страницу аккаунта **Viewed Products** - товары должны отображаться в списке
- На странице товара, где размещён виджет, блок показывает недавно просмотренные товары и исключает текущий

## Использование

Экран настроек (**Расширения -> Модули -> alexwaha.com - Viewed Products**) содержит три вкладки.

### Вкладка Widget (на каждый экземпляр блока)

- **Module Name** - административное название данного экземпляра виджета.
- **Block Title** - заголовок, отображаемый над блоком (по языку). Пусто = стандартное "Viewed Products".
- **Products Limit** - количество товаров, отображаемых в блоке.
- **Image Width / Height** - размер миниатюры.
- **Show "View All" Link** - показывать ссылку на страницу аккаунта в заголовке блока.
- **Status** - включить / отключить данный экземпляр виджета.

### Вкладка Tracking (глобальная)

- **Storage Days** - количество дней хранения истории (одновременно является временем жизни гостевого cookie).
- **Max Stored Products** - максимальное количество товаров, сохраняемых для одного посетителя (старые обрезаются).

### Вкладка Account Page (глобальная)

- **Enable Account Page** - включить / отключить отдельную страницу (при отключении страница возвращает 404).
- **Show Account Menu Link** - добавить ссылку `Viewed Products` в аккаунт покупателя (как в списке на странице аккаунта, так и в модуле меню аккаунта).
- **Menu Link Text** - текст ссылки (по языку). Пусто = стандартное "Viewed Products".
- **SEO Keyword** - ЧПУ-URL для страницы, по магазину и языку (например, `viewed-products`). Оставьте пустым для использования стандартного URL вида `index.php?route=...`. Страница всегда исключена из индексирования (`noindex,follow`).

## Для разработчиков

### Технический обзор

Модуль построен полностью на **системе событий OpenCart** - без модификаций OCMOD. События регистрируются при установке (через `setting/event` для 3.x, `extension/event` для 2.3, выбор происходит автоматически через `awCore->isLegacy()`):

| Триггер | Действие | Назначение |
|---|---|---|
| `catalog/controller/product/product/before` | `extension/module/aw_viewed/track` | Записывает просмотренный товар (один раз за GET-запрос, только для реального включённого товара) |
| `catalog/controller/account/account/before` | `extension/module/aw_viewed/accountLogin` | Объединяет историю из гостевого cookie с аккаунтом покупателя при входе |
| `catalog/view/account/account/after` | `extension/module/aw_viewed/accountList` | Добавляет ссылку в список «Мой аккаунт» на странице аккаунта |
| `catalog/view/extension/module/account/after` | `extension/module/aw_viewed/accountMenu` | Добавляет ссылку в модуль меню аккаунта (боковая панель) |

### Фронтенд (только AJAX)

Виджет и страница аккаунта отображают только оболочку на стороне сервера, а товары загружаются через AJAX:

- Товары виджета: `extension/module/aw_viewed/products`
- Список страницы аккаунта: `extension/module/aw_viewed_page/list`
- Удаление элемента: `extension/module/aw_viewed/delete` (только для авторизованных покупателей)

Оболочка страницы безопасна для кеша полных страниц; данные конкретного посетителя загружаются на клиенте небольшим скриптом на чистом JS (без зависимости от jQuery).

### База данных

- `oc_aw_viewed` - одна строка на каждый просмотренный товар: `session_token` (гостевой cookie), `customer_id`, `product_id`, `store_id`, `date_added`. Дедупликация по посетителю, обрезается до **Max Stored Products**, очищается по **Storage Days**.

Конфигурация модуля хранится в `oc_aw_module_config` (ключ `aw_viewed`); настройки экземпляра виджета используют нативную таблицу `oc_module`; SEO-ключ страницы - нативную таблицу `oc_seo_url`.

### Расположение файлов

- **Контроллер виджета и событий:** `catalog/controller/extension/module/aw_viewed.php`
- **Контроллер страницы аккаунта:** `catalog/controller/extension/module/aw_viewed_page.php`
- **Каталог модели (отслеживание/запросы):** `catalog/model/extension/module/aw_viewed.php`
- **Админ-контроллер / модель:** `admin/.../extension/module/aw_viewed.php`

> **Критическое требование:** Использование модуля возможно только вместе с системным модулем **alexwaha.com - Core**
