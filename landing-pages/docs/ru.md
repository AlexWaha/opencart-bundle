# Alexwaha.com - Посадочные страницы

**Модуль для OpenCart v2.3 - 3.x**

---

## Описание

Это дополнение содержит модули для создания посадочных страниц и модуль для вывода блока ссылок на них.

Модуль позволяет создавать отдельные посадочные страницы с набором товаров, а также выводить блок со ссылками на эти страницы в любом месте сайта.

---

## Инструкция по установке

### Требования и этапы установки

1. **Установка основного модуля**

   Убедитесь в том, что у Вас установлен модуль **[aw_core_oc2.3-3.x.ocmod.zip](https://github.com/AlexWaha/opencart-bundle/blob/master/Core/dist/aw_core_oc2.3_3.x.ocmod.zip)**

   > Это вспомогательный функционал для работы всех моих модулей. **Критически необходимо**, без него работать модуль не будет

2. **Настройки FTP-загрузки (только для OpenCart 2.x)**

   Убедитесь в том, что у Вас установлен модуль отключающий FTP-загрузку дополнений в OpenCart 2.x или у Вас корректные настройки FTP в настройках магазина

   > [Ссылка на модуль отключающий FTP-загрузку только для OpenCart v2.3](https://www.opencart.com/index.php?route=extension/extension/info&extension_id=18892) **(Для OpenCart 3.x не требуется)**

3. **Установите архив aw_landing_page_oc2.3-3.x.ocmod.zip через админ-панель сайта**

4. **Обновите кеш модификаций**

5. **Включение модуля**

   В разделе: **Расширения → Модули** → Включите модуль **alexwaha.com - Посадочные страницы**

   > При включении модуль автоматически установит необходимые таблицы в БД и выдаст права на просмотр и редактирование

### Самопроверка после установки

- Проверить наличие модуля в списке дополнений
- Обновить кеш модификаций

## Настройка

### Создание посадочных страниц

1. В левой колонке появился новый пункт **Каталог → Посадочные страницы**, зайдите в него

   Или через раздел **Дополнения → alexwaha.com - Посадочные страницы**

2. Добавьте новую посадочную страницу заполняя все обязательные поля и добавив товары для вывода *(работает автодополнение)*

### Настройка модуля ссылок

3. Зайдите в раздел **Дополнения → Модули - alexwaha.com - Посадочные страницы/ссылки**

4. Задайте системное **Имя модуля**, **Заголовок блока** и **Страницы** которые нужны для вывода *(работает автодополнение)*, **Статус**, **Сохраните изменения**

5. Зайдите в раздел **Дизайне → Схемы**, выберите схему на которой хотите вывести модуль со ссылками посадочных страниц и его позицию

> Например: Схема **"Главная"**, позиция **"Внизу страницы"** → **Сохраните изменения**

> **Важно:** Модуль будет выводиться на выбранной схеме в виде блока с Заголовком и списком ссылок

> **Примечание:** Ссылки на посадочные страницы автоматически появятся в Карте сайта **sitemap.xml**

## Разработчикам

### Техническая информация

Модуль использует шаблонизатор Twig, а также не требует объявления языковых переменных внутри контроллера, достаточно просто добавить переменную в языковой файл.

Языковые файлы модуля всегда в папках `en-gb`, `ru-ru` вне зависимости от версии OpenCart.

### Расположение файлов

- **Шаблон модуля ссылок:** `../default.../extension/module/aw_langing_links.twig`
- **Шаблон посадочной страницы:** `../default.../extension/module/aw_langing_page.twig`
