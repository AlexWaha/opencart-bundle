Instruction | Alexwaha.com - Landing pages for Opencart v2.3 -3.x

Installation Instructions

This extension includes modules for creating landing pages and a module for displaying a block of links to them

  • 1. Make sure you have installed the module aw_core_oc2.3-3.x.ocmod.zip This is an auxiliary function for all my modules
  • 2. Make sure you have a module installed that disables FTP uploads for extensions in Opencart 2.x, or that your FTP settings in the store config are correct – Link to module disabling FTP upload for Opencart v2.3 only (Not required for Opencart 3.x)
  • 3. Install the archive aw_landing_page_oc2.3-3.x.ocmod.zip via the site's admin panel
  • 4. Refresh the modification cache
  • 6. Go to: Extensions Modules Enable the module – alexwaha.com - Landing Pages When enabled, the module will automatically create the required DB tables and grant view/edit permissions

Self-check after installation:

  • Check for the module in the list of extensions
  • Refresh the modification cache
Configuration

  • 1. A new item Catalog Landing Pages appeared in the left column, open it

    or via Extensions alexwaha.com - Landing Pages

  • 2. Add a new landing page, filling in all required fields and adding products to display (autocomplete works)
  • 3. Go to Extensions Modules - alexwaha.com - Landing Pages/Links
  • 4. Set the system Module Name, Block Title, and Pages to display (autocomplete works), Status, Save changes
  • 5. Go to Design Layouts, choose the layout where you want to display the module with landing page links and its position.

    Example: Layout "Home", position "Bottom of the page" Save changes

The module will appear on the selected layout as a block with a Title and a list of links

Landing page links will automatically appear in the sitemap sitemap.xml

For Developers

The module uses the Twig templating engine, and does not require declaring language variables inside the controller, just add the variable to the language file.

The module's language files are always located in the en-gb, ru-ru folders regardless of the Opencart version.

Template file location for the links module: ../default.../extension/module/aw_langing_links.twig

Template file location for the landing page: ../default.../extension/module/aw_langing_page.twig

Use of the module is only possible together with the system module alexwaha.com - Core

Attention! Technical support for this module is only available on a paid basis. Bug fixes in the main repository are done without a schedule as long as the developer has free time. If you can offer a solution to any problem, a pull-request on GitHub is welcome.

Setting up the module by the author, customization, resolving conflicts with other modules, integration with a template, integration with another module in your project - only on a paid basis, please contact us below.

Tested on a clean Opencart and on the default theme (Default).

Инструкция | Alexwaha.com - Посадочные страницы для Opencart v2.3 - 3.x

Инструкция по установке

Это дополнение содержит модули для создания посадочных страниц и модуль для вывода блока ссылок на них

  • 1. Убедитесь в том, что у Вас установлен модуль aw_core_oc2.3-3.x.ocmod.zip Это вспомагательный фунционал для работы всех моих модулей
  • 2. Убедитесь в том, что у Вас установлен модуль отключающий ftp-загрузку дополнений в Opencart 2.x или у Вас корректные настройки FTP в настройках магазина - Ссылка на модуль отключающий ftp-загрузку только для Opencart v2.3 (Для Opencart 3.x не требуется)
  • 3. Установите архив aw_landing_page_oc2.3-3.x.ocmod.zip через админ-панель сайта
  • 4. Обновите кеш модификаций
  • 6. В разделе: Расширения Модули Включите модуль - alexwaha.com - Посадочные страницы При включении модуль автоматически установит необходимые таблицы в БД и выдаст права на просмотр и редактирование

Самопроверка после установки:

  • Проверить наличие модуля в списке дополнений
  • Обновить кеш модификаций
Настройка

  • 1. В левой колонке появился новый пункт Каталог Посадочные страницы, зайдите в него

    или через раздел Дополнения alexwaha.com - Посадочные страницы

  • 2. Добавьте новую посадочную страницу заполняя все обязательные поля и добавив товары для вывода (работает автодополнение)
  • 3. Зайдите в раздел Дополнения Модули - alexwaha.com - Посадочные страницы/ссылки
  • 4. Задайте системное Имя модуля, Заголовок блока и Страницы которые нужны для вывода (работает автодополнение), Статус, Сохраните изменения
  • 5. Зайдите в раздел Дизайне Схемы, выберите схему на которой хотите вывести модуль со ссылками посадочных страниц и его позицию.

    Например: Схема "Главная", позиция "Внизу страницы" Сохраните изменения

Модуль будет выводиться на выбранной схеме в виде блока с Заголовком и списком ссылок

Ссылки на посадочные страницы автоматически появятся в Карте сайта sitemap.xml

Разработчикам

Модуль использует шаблонизатор Twig, а также не требует обьявления языковых переменных внутри контроллера, достаточно просто добавить переменную в языковой файл.

Языковые файлы модуля всегда в папках en-gb, ru-ru вне зависимости от версии Opencart.

Расположение файла шаблона для модуля ссылок в папке ../default.../extension/module/aw_langing_links.twig

Расположение файла шаблона для посадочной страницы в папке ../default.../extension/module/aw_langing_page.twig

Использование модуля возможно только совместно с системным модулем alexwaha.com - Core
