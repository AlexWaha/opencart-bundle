# Alexwaha.com - Глобальный макет

**Модуль для OpenCart v2.3 - 3.x**

---

## Описание

Данный модуль создает один глобальный макет, в который можно добавить модули, и эти модули будут отображаться на всех страницах вашего магазина OpenCart.

Модуль Глобальный Макет позволяет размещать модули в едином макете, которые будут автоматически отображаться на каждой странице вашего интернет-магазина.

---

## Инструкция по установке
### Требования и этапы установки

1. **Установка основного модуля**

   Убедитесь в том, что у Вас установлен модуль **[aw_core_oc2.3-3.x.ocmod.zip](https://github.com/AlexWaha/opencart-bundle/blob/master/Core/dist/aw_core_oc2.3_3.x.ocmod.zip)**

   > Это вспомогательный функционал для работы всех моих модулей. 
   
   > **Критически необходимо**, без него работать модуль не будет

2. **Настройки FTP-загрузки (только для OpenCart 2.x)**

   Убедитесь в том, что у Вас установлен модуль отключающий FTP-загрузку дополнений в OpenCart 2.x или у Вас корректные настройки FTP в настройках магазина

   > [Ссылка на модуль отключающий FTP-загрузку только для OpenCart v2.3](https://www.opencart.com/index.php?route=extension/extension/info&extension_id=18892) **(Для OpenCart 3.x не требуется)**

3. **Установите файлы модуля aw_global_layout_oc2.3-3x.ocmod.zip:**
4. **Обновите кеш модификаций**
5. **Включение модуля**

   В разделе: **Расширения → Модули** → Включите модуль **alexwaha.com - Глобальный макет**

   > При включении модуль автоматически выдаст права на просмотр и редактирование, а также создаст необходимые таблицы базы данных

### Самопроверка после установки

- Проверить наличие модуля в списке дополнений
- Обновить кеш модификаций
- Убедиться, что интерфейс глобального макета доступен
- Проверить позиционирование модулей на сайте

## Настройка

### Параметры модуля

1. Зайдите в раздел **Дополнения → Модули → alexwaha.com - Глобальный макет**

2. Настройте параметры модуля:

   - **Название макета:** Задайте имя для глобального макета (по умолчанию: "All pages")
   - **Статус:** Включить/Отключить глобальный макет
   - **Добавление модулей:** Добавляйте модули в глобальный макет из выпадающих списков
   - **Порядок сортировки:** Перетаскивайте модули за ручку для изменения их порядка внутри одной позиции

3. Сохраните конфигурацию

4. При включении все модули в глобальном макете будут отображаться на каждой странице магазина

> **Важно:** Глобальный макет отображает свои модули на всех страницах при активном макете

> **Примечание:** Модуль поддерживает перетаскивание для изменения порядка модулей внутри каждой позиции с помощью визуальных ручек

### Использование модуля

Модуль Глобальный Макет создает единый глобальный макет, в который можно добавить модули, и они будут отображаться на всех страницах магазина.

1. **Доступ к модулю:** Перейдите в Дополнения → Модули → Глобальный макет
2. **Выбор модулей:** Выберите любой установленный модуль из выпадающих списков
3. **Добавление в макет:** Добавьте модули в позиции глобального макета (Верх/Низ контента, Левая/Правая колонки, Полная ширина Верх/Низ)
4. **Изменение порядка модулей:** Используйте ручку перетаскивания для изменения порядка модулей только внутри одной позиции
5. **Удаление модулей:** Нажмите кнопку удаления для удаления модулей из позиций
6. **Сохранение конфигурации:** Нажмите "Сохранить" чтобы модули появились на всех страницах

### Доступные позиции макета

- **Полная ширина Верх** - охватывает всю ширину страницы
- **Верх контента** - выше основной области контента
- **Левая колонка** - левая боковая панель
- **Правая колонка** - правая боковая панель
- **Низ контента** - ниже основной области контента
- **Полная ширина Низ** - охватывает всю ширину страницы

> **Важно:** Модули, добавленные в глобальный макет, будут отображаться на всех страницах магазина. Удалите модули из макета, если не хотите их показывать на всех страницах.

## Разработчикам

### Техническая информация

Модуль использует шаблонизатор Twig, а также не требует объявления языковых переменных внутри контроллера, достаточно просто добавить переменную в языковой файл.

Языковые файлы модуля всегда в папках `en-gb`, `ru-ru` вне зависимости от версии OpenCart.

### Расположение файлов

- **Шаблон админки:** `admin/view/template/extension/module/aw_global_layout.twig`
- **Шаблоны позиций:** `catalog/view/theme/default/template/extension/aw_position/aw_content_top.twig`, `aw_content_bottom.twig`

### Основные функции

- `getLayout($route)` - Получает конфигурацию макета для определённого маршрута
- `getModules($id)` - Получает все модули, назначенные позиции макета
- `editLayout($id, $data)` - Обновляет конфигурацию макета и назначения модулей
- `install($name, $route)` - Создаёт таблицы базы данных и начальный макет
