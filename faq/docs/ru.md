## 2. Ключевые возможности

- Полный CRUD для вопросов и ответов
- Визуальный редактор Summernote для форматирования ответов
- Отдельная страница /faq/ с Bootstrap-аккордеоном (иконки +/-)
- SEO URL и мета-теги для каждого языка
- Интеграция в меню админки через OCMOD
- Управление порядком сортировки
- Импорт/Экспорт настроек в формате JSON
- Мультиязычность (en-gb, ru-ru, uk-ua)

## 3. Установка

### Предварительные требования

- OpenCart 2.3 - 3.x
- Установленная библиотека AwCore
- PHP 7.4+

### Шаги установки

1. Перейдите в **Расширения - Установщик расширений**
2. Загрузите `aw_faq_oc2.3-3.x.ocmod.zip`
3. Перейдите в **Расширения - Модификации** и нажмите **Обновить**
4. Перейдите в **Расширения - Модули**
5. Найдите **alexwaha.com - FAQ** и нажмите **Установить** (создаст таблицы БД)
6. Нажмите **Редактировать** для настройки и добавления вопросов

## 4. Настройка

### 4.1 Список FAQ

Главная страница модуля показывает все вопросы в таблице: вопрос, порядок сортировки, статус, действия (редактировать/удалить). Кнопка **Добавить** создаёт новый вопрос.

### 4.2 Форма FAQ

При добавлении или редактировании вопроса:

- **Вопрос** - Текст вопроса (для каждого языка)
- **Ответ** - Форматированный текст с редактором Summernote (для каждого языка). Поддерживает изображения, форматирование, ссылки через встроенный файловый менеджер.
- **Порядок сортировки** - Числовой порядок для отображения
- **Статус** - Включить или выключить вопрос

### 4.3 Вкладка "Настройки"

- **Статус** - Глобальный статус модуля
- **SEO URL** - Ключевое слово URL для страницы FAQ (для каждого языка)
- **Мета-заголовок** - Заголовок страницы для SEO (для каждого языка)
- **Мета-описание** - Описание страницы для SEO (для каждого языка)

### 4.4 Импорт / Экспорт

- **Экспорт** - Скачать настройки FAQ в формате JSON
- **Импорт** - Загрузить ранее экспортированный JSON

Контактная информация для поддержки модуля.

## 5. Решение проблем

| Проблема | Решение |
|----------|---------|
| Страница FAQ выдаёт 404 | Задайте SEO URL в настройках, очистите кеш SEO |
| Редактор Summernote не загружается | Проверьте что стили AwCore подключены |
| FAQ не отображается на сайте | Проверьте статус модуля и статус отдельных вопросов |
| Меню OCMOD не появляется | Обновите кеш модификаций |
| Изображения в ответах не работают | Используйте файловый менеджер внутри Summernote |
