# Alexwaha.com - Калькулятор разбавления самогона

**Модуль для OpenCart v2.3 - 3.x**

---

## Описание

Данный модуль предоставляет калькулятор разбавления самогона, который автоматически рассчитывает количество воды, необходимое для получения желаемой крепости алкоголя.

Калькулятор помогает определить точное количество воды для разбавления самогона до нужной крепости с учетом изначального объема и процента алкоголя.

---

## Инструкция по установке

### Требования и этапы установки

1. **Установка основного модуля**

   Убедитесь в том, что у Вас установлен модуль **[aw_core_oc2.3-3.x.ocmod.zip](https://github.com/AlexWaha/opencart-bundle/blob/master/Core/dist/aw_core_oc2.3_3.x.ocmod.zip)**

   > Это вспомогательный функционал для работы всех моих модулей.

   > **Критически необходимо**, без него работать модуль не будет

2. **Настройки FTP-загрузки (только для OpenCart 2.x)**

   Убедитесь в том, что у Вас установлен модуль отключающий FTP-загрузку дополнений в OpenCart 2.x или у Вас корректные настройки FTP в настройках магазина

   > [Ссылка на модуль отключающий FTP-загрузку только для OpenCart v2.3](https://www.opencart.com/index.php?route=extension/extension/info&extension_id=18892) **(Для OpenCart 3.x не требуется)**

3. **Установите файлы модуля aw_moonshine_calculator_oc2.3-3.x.ocmod.zip**

4. **Обновите кеш модификаций**

5. **Включение модуля**

   В разделе: **Расширения → Модули** → Включите модуль **alexwaha.com - Калькулятор самогона**

   > При включении модуль автоматически выдаст права на просмотр и редактирование

### Самопроверка после установки

- Проверить наличие модуля в списке дополнений
- Обновить кеш модификаций
- Проверить работоспособность калькулятора на сайте

## Настройка

### Параметры модуля

1. Зайдите в раздел **Дополнения → Модули → alexwaha.com - Калькулятор самогона**

2. Настройте параметры модуля:

   - **Статус:** Включить/Отключить калькулятор
   - **Заголовок страницы:** Задайте заголовок страницы для каждого языка
   - **Заголовок H1:** Задайте основной заголовок для каждого языка
   - **Описание:** Добавьте описание страницы (поддерживает HTML)
   - **Инструкция:** Добавьте инструкцию по использованию (поддерживает HTML)
   - **SEO URL:** Задайте дружественный URL (по умолчанию: moonshine-calculator)

3. Сохраните конфигурацию

4. Калькулятор будет доступен по адресу: `ваш-сайт.com/moonshine-calculator` (или ваш SEO URL)

> **Важно:** Страница калькулятора автоматически добавляется в карту сайта **sitemap.xml**

> **Примечание:** Модуль поддерживает AJAX-расчеты для мгновенного получения результатов без перезагрузки страницы

---

### Использование калькулятора

Калькулятор помогает определить, сколько воды добавить при разбавлении самогона для получения желаемой крепости.

1. **Начальная крепость (%):** Введите текущую крепость алкоголя (0-100%)
2. **Желаемая итоговая крепость (%):** Введите целевую крепость алкоголя (должна быть меньше начальной)
3. **Объем самогона (л):** Введите объем самогона для разбавления
4. **Нажмите "Рассчитать"** для получения результатов

### Результаты расчета

- Количество воды для добавления (л)
- Итоговый общий объем (л)

> **Важно:** Все расчеты актуальны для температуры компонентов 20°C

## Разработчикам

### Техническая информация

Модуль использует шаблонизатор Twig, а также не требует объявления языковых переменных внутри контроллера, достаточно просто добавить переменную в языковой файл.

Языковые файлы модуля всегда в папках `en-gb`, `ru-ru` вне зависимости от версии OpenCart.

### Расположение файлов

- **Шаблон каталога:** `catalog/view/theme/default/template/extension/module/aw_moonshine_calculator.twig`
- **CSS стили:** `catalog/view/javascript/aw_moonshine_calculator.min.css`

### Основные функции

- `calculate($initial_strength, $final_strength, $moonshine_volume)` - Основной метод расчета
- `getTemperatureCorrection($temperature)` - Коррекция по температуре (для будущих обновлений)
- `validateStrength($strength)` - Проверка корректности крепости
- `validateVolume($volume)` - Проверка корректности объема
