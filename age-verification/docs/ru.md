# Alexwaha.com - Подтверждение возраста

**Модуль для OpenCart v2.3 - 3.x**

---

## Описание

Данный модуль предоставляет возрастную верификацию (18+) в виде всплывающего окна, которое появляется при первом посещении сайта для ограничения доступа несовершеннолетних посетителей.

Всплывающее окно верификации возраста отображается при первом посещении магазина пользователями, требуя подтверждения совершеннолетия перед доступом к содержимому сайта.

---

## Инструкция по установке

### Требования и этапы установки

1. **Установка основного модуля**

   Убедитесь в том, что у Вас установлен модуль **[aw_core_oc2.3-3.x.ocmod.zip](https://github.com/AlexWaha/opencart-bundle/blob/master/Core/dist/aw_core_oc2.3_3.x.ocmod.zip)**

   > Это вспомогательный функционал для работы всех моих модулей. **Критически необходимо**, без него работать модуль не будет

2. **Настройки FTP-загрузки (только для OpenCart 2.x)**

   Убедитесь в том, что у Вас установлен модуль отключающий FTP-загрузку дополнений в OpenCart 2.x или у Вас корректные настройки FTP в настройках магазина

   > [Ссылка на модуль отключающий FTP-загрузку только для OpenCart v2.3](https://www.opencart.com/index.php?route=extension/extension/info&extension_id=18892) **(Для OpenCart 3.x не требуется)**

3. **Установите файлы модуля aw_age_verification_oc2.3-3.x.ocmod.zip**
4. **Обновите кеш модификаций в панели администрирования OpenCart**
5. **Включение модуля**

   В разделе: **Расширения → Модули** → Включите модуль **alexwaha.com - Подтверждение возраста**

   > При включении модуль автоматически выдаст права на просмотр и редактирование

### Самопроверка после установки

- Проверить наличие модуля в списке дополнений
- Обновить кеш модификаций
- Проверить работу всплывающего окна на сайте (предварительно очистив куки)

## Настройка

### Параметры модуля

1. Зайдите в раздел **Дополнения → Модули → alexwaha.com - Подтверждение возраста**

2. Настройте параметры модуля:

   - **Статус:** Включить/Отключить всплывающее окно верификации возраста
   - **Заголовок блока:** Задайте заголовок всплывающего окна для каждого языка (например, "Вам исполнилось 18 лет?")
   - **Описание:** Добавьте текст описания для каждого языка (поддерживает визуальный редактор)
   - **Срок хранения ответа (дни):** Задайте, как долго помнить ответ пользователя (1-365 дней, по умолчанию: 30)
   - **URL перенаправления:** URL для перенаправления пользователей, отклонивших верификацию

3. Сохраните конфигурацию

4. Всплывающее окно будет автоматически появляться на всех страницах, пока пользователь не примет его

> **Важно:** Всплывающее окно автоматически интегрируется во все страницы через систему модификаций OCMOD

> **Примечание:** Ответ пользователя сохраняется в куки и запоминается на настроенный период

---

### Процесс взаимодействия с пользователем

1. **Первое посещение:** Появляется всплывающее окно по центру экрана с темным фоном
2. **Выбор пользователя:** Посетитель нажимает "Да" (мне есть 18) или "Нет" для подтверждения возраста
3. **Принятие:** Если "Да", окно исчезает и устанавливается кука на настроенный период
4. **Отклонение:** Если "Нет", пользователь перенаправляется на указанный URL (если настроен)
5. **Повторные посещения:** Окно больше не показывается до истечения срока куки

### Особенности

- Адаптивный дизайн, работающий на всех устройствах
- Плавная анимация и профессиональное оформление
- Поддержка многоязычности
- Настраиваемый срок хранения куки
- Опциональное перенаправление при отклонении верификации

> **Важно:** Это базовая система верификации возраста и не должна рассматриваться как надежный метод ограничения доступа по возрасту

## Разработчикам

### Техническая информация

Модуль использует шаблонизатор Twig, а также не требует объявления языковых переменных внутри контроллера, достаточно просто добавить переменную в языковой файл.

### Расположение файлов

- **Файл шаблона:** `catalog/view/theme/default/template/extension/module/aw_age_verification.twig`
- **CSS стили:** `catalog/view/javascript/aw_age_verification.css`

### Ключевые компоненты

- **Интеграция OCMOD** - Автоматическая интеграция в footer без изменения шаблонов
- **Управление куки** - Сохранение ответа пользователя с настраиваемым сроком
- **AJAX обработка** - Плавное взаимодействие без перезагрузки страницы
- **Адаптивный дизайн** - Мобильно-дружественное окно с центрированием на flexbox
- **Визуальный редактор** - Интеграция Summernote для полей описания

> **Критическое требование:** Использование модуля возможно только совместно с системным модулем **alexwaha.com - Core**
